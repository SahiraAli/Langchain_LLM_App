# Blog Title Generator (LangChain LLM App)

This repository contains a tutorial-based LangChain application demonstrating:

- Prompt templates
- Async LLM pipelines
- Helper abstractions
- Async pytest-based testing

The project is structured to follow best practices for environment isolation and secret management.

├── langchain_helper.py
├── test_pipeline_async.py
├── requirements.txt
